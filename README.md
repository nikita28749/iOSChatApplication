# Instant Chat App
Instant Chat App is an iOS based mobile app that allows users to communicate with each other by sending instant messages or by making video calls. 

## Technologies Used
 * XCode
 * Parse DB
 * Amazon AWS
 * Heroku 

## Features
 * User Registration/Login
 * Instant messaging
 * App to Phone Call
 * App to App Call
 * Facetime Call
 * Video Call
